ecma模块

描述：

本模块是用于扩展javascript语言能支持大多数ECMA262v5 API。因此对于像chrome，FF3，IE9，opera10,safari4等较新的浏览器， dom Framework的模块加载器会通过特征嗅征决定是否加载它，而这步工作是在语言扩展模块-lang中进行, 因此在$.require中的依赖列表中，我们只需指定lang模块。

本模块将为javascript添加如下API（具体用法请参考MSDN或MDC）：

  • Object.keys
  • Object.getPrototypeOf
  • Array.prototype.indexOf
  • Array.prototype.lastIndexOf
  • Array.prototype.forEach
  • Array.prototype.filter
  • Array.prototype.map
  • Array.prototype.some
  • Array.prototype.every
  • Array.prototype.reduce
  • Array.prototype.reduceRight
  • Array.isArray
  • String.prototype.trim
  • Function.prototype.bind
  • Date.now

修复IE下的三个实现错误

  • Array.prototype.unshift
  • Date.prototype.getYear
  • Date.prototype.setYear
