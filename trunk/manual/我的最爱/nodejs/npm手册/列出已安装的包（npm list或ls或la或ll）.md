# 列出已安装的包（npm list/ls/la/ll） #
By Tom.Wang<tom.wang.sz@qq.com\>

## 概述 ##
1、原始文档地址：[https://npmjs.org/doc/list.html](https://npmjs.org/doc/list.html)    

## 语法 ##
> `npm list [<pkg...]`  
> `npm ls [<pkg...]`    
> `npm la [<pkg...]`  
> `npm ll [<pkg...]`  

1、不带参数，列出所有已安装的包  
2、带上参数，列出相关的已安装的包  
3、la和ll显示的信息更详细一些
