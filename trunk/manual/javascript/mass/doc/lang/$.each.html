$.each(obj, callback, [ args ])

描述：

用于遍历类数组与普通对象。

回调函数的传参情况：

  1. 如果第三个参数不是数组，并且obj为类数组，那么依次为数组元素，当前索引值与类数组本身。
  2. 如果第三个参数是数组，并且obj为类数组，那么参数就是第三个参数里面的元素。
  3. 如果第三个参数不是数组，并且obj为对象，那么依次为值，键名与对象本身。
  4. 如果第三个参数是数组，并且obj为对象，那么参数就是第三个参数里面的元素。

如果回调函数返回false,会立即中断遍历。

回调函数中的this始终为当前元素（obj为类数组时）或值（obj为对象时）。

返回值：

无。

例子 
                $.require("lang",function(){
                    $.each({
                        aa:"人生",
                        bb:"工作",
                        cc:"评议",
                        dd:"祈祷"
                    },function(){
                        $.log(this, true);
                        if(this === "评议");
                        return false;
                    });
                    $.each({
                        aa:"人生",
                        bb:"工作"
                    },function(a,b){
                        $.log(a, true);
                        $.log(b, true);
                    },[1,2]);
                    $.each(["Wii U"," PS Vita", "Xperia"],function(el){
                        $.log(el, true);
                    });
                });
