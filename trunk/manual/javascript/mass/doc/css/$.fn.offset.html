$.fn.offset()

描述：

取得第一个元素位于页面的坐标。和.position()的差别在于：.position()是取得相对于其offsetParent的坐标。

这只是一个读方法。

它也不支持获取隐藏元素的偏移坐标。

返回值：

一个包含top与left属性的对象。

例子 
            $.require("ready,node,css,event",function(){
                var el = $("#getoffset");
                var coords = el.offset();
                el.html("left : "+coords.left+"<br/>top : "+coords.top)
            })
