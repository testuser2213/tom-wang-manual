$.fn.position()

描述：

取得第一个元素相对其offsetParent的坐标。

这只是一个读方法。

它也不支持获取隐藏元素的偏移坐标。

返回值：

一个包含top与left属性的对象。

例子 
            $.require("ready,node,css,event",function(){
                var el = $("#getpositon");
                var coords = el.position();
                el.html("left : "+coords.left+"<br/>top : "+coords.top)
            })
